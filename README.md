## Kurulum

1. Bu depoyu klonlayın:
    ```bash
    git clone https://github.com/Abdulkadirkilicc/DepoMaECommerce-Solution.git
    ```

2. Proje dizinine gidin:
    ```bash
    cd DepoMaECommerce-Solution
    ```

3. Gerekli bağımlılıkları yükleyin:
    ```bash
    npm install
    ```

## Çalıştırma

1. Uygulamayı başlatın:
    ```bash
    npm start
    ```
